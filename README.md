## 📌 Project Overview

This project is a responsive web page built using **vanilla HTML, CSS, and JavaScript** (no frameworks or libraries), following the exact design specifications provided in the Figma file.

The implementation ensures pixel-perfect accuracy, responsiveness across devices, and clean, maintainable code.

## 🚀 Features Implemented

### ✅ Sticky Header
- Appears when scrolling beyond the first fold
- Positions above the navigation bar
- Disappears when scrolling back up
- Smooth transition and animation effects

### ✅ Image Carousel with Zoom
- Interactive image carousel
- Hover-based zoom preview functionality
- Smooth hover effects and animations
- Fully responsive behavior

### ✅ Fully Responsive Design
- Desktop
- Tablet
- Mobile

### Clean Code Standards
- Semantic HTML5 structure
- Modern CSS (Flexbox & Grid)
- Organized file structure
- Commented JavaScript
- Cross-browser compatibility
- Performance optimized
